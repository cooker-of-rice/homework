# Operace Cerberus

- spočívá v srovnání dat podle toho kolik času zaberou na srovnání stažení (pod 300s - čas srovnání výsledků) pomocí hladového algoritmu 

1. zformátovat data a dát je do objektu
2. výpočet efektivity čas stažení / kredity
3. zarovnat je za sebou a algoritmus bude vybírat od zhora
4. to co se vejde do časového limitu tak dáme na výstup
